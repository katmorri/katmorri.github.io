Moog Ladder Filter

Designed by Robert Moog in 1966, the Moog Voltage-Controlled Filter (VCF)—or Moog ladder filter—is a key element of the Moog synthesizer sound and a classic piece of electronic music history. In addition to its popularity today, the filter found its place on a number of famous records from the 1970s and 80s, including Kraftwerk's The Man Machine, Bob Marley's Catch a Fire, and Michael Jackson's Thriller, to name a few.

The Moog VCF is a set of four low-pass filters connected by 10 transistors. I implemented a linear model of the circuit using the state-space modeling technique. The method leads to a system of differential equations that I solved using three different approaches: forward, backward, and trapezoidal integration. A real-time C++ implemenation of the trapezoidal integration algorithm is available to download for free on my GitHub page. The page also includes instructions for compiling the source code and running the VST plugin in a DAW.

Vacuum Tube Distortion

Vacuum tube distortion is a vintage audio effect found in many old music amplifiers, especially guitar amplifiers. One well-known guitar amplifier that produces this type of distortion is the Fender Bassman (shown at right). The effect is caused by a valve-like clipping that adds additional harmonics to the original input audio.

I modeled the valve-like clipping using an approach based on the work of R. Bendiksen. The main equation of the model takes the form $$y = {x - Q \over 1 - exp(-D(x-Q))} + {Q \over 1 - exp(DQ)},$$ where \(x \) is the original input audio, \(y \) is the distorted output audio, \(Q \) is a parameter that determines the linearity of the clipping function, and \(D \) defines the harshness of the distortion.
